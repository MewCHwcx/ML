OV & MIF Classification

This is a demonstration of image classification using a model trained with Google's Teachable Machine project. If you cover the camera, this model will classify the image as "OV", otherwise will classify anything else as "MIF"

OV (Opisthorchis Viverrini)

  • ขนาด
    👉 25 x 15 ไมครอน
  • ลักษณะ
    👉 คล้ายเหยือกน้ำหรือหลอดไฟ ผิวขรุขระ
  • ส่วนประกอบ
    👉 มีปุ่ม(Knob), บ่า(Shhoulder), ฝา(Operculum)

MIF (Minute Intestinal Fluke)

  • ขนาด
    👉 ใกล้เคียงกับ OV
  • ลักษณะ
    👉 ผิวเรียบ
  • ส่วนประกอบ
    👉 ส่วนใหญ่ไม่มีปุ่ม(Knob), มีบ่า(Shhoulder), มีฝา(Operculum)
